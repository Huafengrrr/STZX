🌟 声途智行 (ShengTuZhiXing) APP v1.0一款基于计算机视觉与深度学习的安卓无障碍出行辅助应用（智能盲杖），致力于用科技为视障人士提供更安全的出行体验。📖 项目简介“声途智行”是一款运行在 Android 平台上的智能辅助出行 APP。本项目利用终端侧的人工智能技术，将智能手机转化为视障人士的“智能盲杖”。通过实时识别前方道路障碍物，并动态计算障碍物的相对距离与移动速度，辅助用户感知周围环境，保障出行安全。✨ 核心功能👁️ 实时目标检测：内置深度学习大模型 (YOLOv8)，利用 TensorFlow Lite 在移动端进行推理，能够精准识别道路上的行人、车辆及各类常见障碍物。📏 智能测距 (优化中)：通过视觉算法估算障碍物与用户之间的物理距离，提供前置安全预警。🏃‍♂️ 实时测速 (优化中)：动态监测移动障碍物的相对速度，判断潜在碰撞风险。💻 纯本地化运行：核心 AI 模型 (yolo.tflite) 完全部署在本地，无需依赖网络环境即可进行实时演算，保证了极低的延迟和高度的隐私保护。🛠️ 技术栈开发环境：Android Studio核心语言：Kotlin / Java, Android XMLAI 引擎：TensorFlow Lite底层模型：YOLOv8 (74MB)版本控制：Git, Git LFS (Large File Storage)⚠️ 当前版本已知问题 (v1.0)作为 1.0 初始版本，目前核心架构已搭建完毕，但在复杂环境下的软件测试中发现以下问题，将在后续版本中持续迭代修复：[Bug] 测距功能：在光线复杂或目标边缘模糊时，距离测算偶尔不够稳定，存在数据跳动。[Bug] 测速功能：在特定场景（如目标快速横向移动）下，测速数据有时无法正常显示。🚀 快速开始 (安装与运行)1. 环境要求Android Studio (建议最新版本)Android 物理设备 (需支持摄像头，推荐 Android 10.0 及以上系统)🚨 极其重要：由于本项目包含 74MB 的 AI 模型文件，克隆代码前请务必确保本地已安装 Git LFS。2. 克隆项目请在终端中依次执行以下命令：# 1. 安装并初始化 Git LFS（如已安装可跳过）
git lfs install

# 2. 克隆仓库到本地
git clone [https://github.com/Huafengrrr/STZX.git](https://github.com/Huafengrrr/STZX.git)
3. 模型配置检查克隆完成后，请务必检查 app/src/main/assets/ 目录下是否存在 yolo.tflite 文件，且大小约为 74MB。(如果该文件只有几 KB，说明 LFS 拉取失败，请在终端执行 git lfs pull 手动拉取，或直接从 GitHub Releases / 网页端下载该模型放入对应目录。)4. 编译与运行使用 Android Studio 打开本项目目录。等待 Gradle Sync 构建完成。连接你的 Android 手机，点击 Run (绿色三角形) 部署到设备即可体验。📅 后续迭代计划 (TODO)[ ] 算法优化：修复测距和测速的 Bug，提升复杂场景下的识别稳定率。[ ] 前端交互体验：优化前端 UI 界面布局，完善无障碍读屏器 (TalkBack) 的适配。[ ] 软件测试：引入更完善的自动化测试用例，覆盖更多极端环境下的边缘测试。[ ] 性能调优：进一步压缩 TFLite 模型体积，降低长时间运行带来的设备发热和功耗。由 Huafengrrr 开发与维护。欢迎提交 Issue 或 Pull Request 共同探讨改进方案！
